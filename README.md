To determine the diameter of the circle `D1 сylinder top and D2 сylinder bottom` you need to $D = { C \over π}$

> `D` = cylinder diameter
> 
> `C` = length of the cylinder circumference
>
> `π` ≈ 3.14159… (3.14)
